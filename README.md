# SQL Injection Practical Exploitation

## Objective
Perform SQL Injection testing using Kali Linux and SQLMap to:
- Identify injectable parameters
- Enumerate databases
- Extract tables and user data
- Analyze security impact
- Suggest mitigation techniques

## Tools Used
- Kali Linux (VMware)
- SQLMap
- DVWA vulnerable lab

## Attack Flow
1. Setup vulnerable lab
2. Identify injectable parameter
3. Run SQLMap detection
4. Enumerate databases
5. Extract tables
6. Dump sensitive data
7. Analyze impact
8. Suggest fixes
